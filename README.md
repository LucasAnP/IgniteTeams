### Getting Started

Clone the project repository

[https://github.com/LucasAnP/IgniteTeams.git](https://github.com/LucasAnP/IgniteTeams.git)

Access the project directory

```
cd IgniteTeams
```

Install the dependencies

```
yarn
  or
npm install
```

Start the APP

```
yarn android
  or
npx expo start

```
